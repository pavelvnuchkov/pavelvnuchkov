* Название проекта: Сетевой чат<br>
Задачи: нужно разработать два приложения для обмена текстовыми сообщениями по сети с помощью консоли (терминала) между двумя и более пользователями.<br>
Первое приложение - сервер чата, должно ожидать подключения пользователей и передачи сообщений всем пользователям подключенным к серверу.<br>
Второе приложение - чат клиента, должно отправлять сообщения введенные с консоли и принимать сообщения с сервера.<br>
Используемые технологии: Многопоточность.<br>
